Read data from the file opened on a foreign host.

The function stops reading characters when end-of-file is reached or when a newline character is read, or when size-1 characters have been read, whichever comes first. The newline character is not discarded. A null character is placed immediately after the last character read into the array

int etb_host_file_open ( ETB *  etb,
const char *  filename,
char *  mode,
dword  flag,
ETB_HOST_FILE **  filep 
)

Open (and create) a file on a foreign host.

Used by Customer Software Module whose file system is user SD card managed by UltimET advanced firmware. As EDI does not currently provide any dirctory management, the user SD is seen as a single directory. Therefore all files will be created in root directory.

Available on UltimET Advanced Customer software module ONLY.

Parameters
etbthe UltimET advanced bus
filenamethe name of the file. No path are allowed
modethe mode in which the file will be opened
  • r Open in read mode
  • w Open in write mode (if the file already exists, its content will be erased)
  • a Open in append mode (write at the end of the file)
flagset of flag which can modify the host file functionalities
  • DSA_HOST_FILE_AUTOFLUSH_FLAG allows to force flush of file at each write
filepa pointer on pointer which will contain the opened file

a negative value if an error or a timeout has occured

◆ etb_host_file_printf()

int etb_host_file_printf ( ETB *  etb,
ETB_HOST_FILE *  filep,
const char *  format,
  ... 
)

Write into a file opened on a foreign host.

If file has been opened with DSA_HOST_FILE_AUTOFLUSH_FLAG flag, the content of the buffer will be really written in the file. This option slow down the write process but garants that the data are transmitted to the UA firmware and written in the file buffer. Otherwise, the data will be written in the file only when file buffer (4KB) is full or during file close operation.
The generated log will be truncated to 4095 characters.

Available on UltimET Advanced Customer software module ONLY.

etbthe ultimET advanced bus
