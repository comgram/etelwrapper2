int  custom_accept_port,
ebool *  connected 
)

Allow to check if the custom created connection is in used.

When etne_start_custom is called, it allows another application to connect to the specified port. If this other application is connected this function will indicate it in the connected out parameter.

Parameters
busthe actually opened bus
custom_accept_portcustom port specified in etne_start_custom
connecteda boolean which will receive "true" if another application is connected to this custom port
