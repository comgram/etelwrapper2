◆ esd_translate_error()

char_cp esd_translate_error ( int  code)

Convert an error code into a textual error message.

Each function of the ESD library returns an integer (int) which represents the error code. If the operation ends with success, the function returns 0, otherwise it returns the number of the error represented by a negative number. This function enables the conversion of this error code into a textual error message in the form of a character string.

Parameters
codeThe error code returned by a dsa lib function.
Returns
A pointer to a string containing the error message.
