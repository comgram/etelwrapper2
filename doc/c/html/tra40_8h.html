Detailed Description

EDI-Version : 4.25A This header file contains public declaration of traduction library.


This library is able to convert a high-level ETEL language command directly into ebl records and vice-versa.
This library is also able to upload registers and sequences from drive into a text file and download sequences or registers text-file into the drives.
This library in conformed to POSIX 1003.1c
