◆ dsa_scale_mapping_activate()

int dsa_scale_mapping_activate ( DSA_DRIVE *  drv,
dword  mode 
)

Activate the scaling which is present in the drive.

A check is done to verify that a mapping area is defined in the drive.

Parameters
drvthe drive whose scaling must be activated
modeindicated if the activation must be cyclic or linear. If cyclic activation is needed, the scaling data should represent a whole turn of the axis (KL27 = KL167)
  • DSA_SCALE_MAPPING_LINEAR_ACTIVATION the activation is linear
  • DSA_SCALE_MAPPING_CYCLIC_ACTIVATION the activation is cyclic
Returns
a negative value if an error or a timeout has occured.
DSA_EACQDEVINUSE One of the device is already doing an acquisition
DSA_EACQNOTPOSSIBLE Drives must be connected with transnet
DSA_EAXESRECOVER Transnet established but axes not recovered
