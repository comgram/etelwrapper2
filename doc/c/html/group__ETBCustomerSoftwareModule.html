◆ etb_csm_cmd_interface_set_fct()

int etb_csm_cmd_interface_set_fct ( ETB *  etb,
unsigned int  fctIdx,
ETB_SHM_CSM_FUNCTION  fct,
void *  user_param 
)

Attach the user function the the specified function index !!! Only available for Customer Software Module running on UltimET Advanced product After that, a user will be able to start the specified function by calling *JMP=fctIdx,threadIdx By calling *JMP=fctIdx,threadIdx, the function will then be executed inside the thread referenced by threadIdx If the specified thread already executes a function, the first function will be ended first, and replaced by the new one.

The function prototype has a first parameter allowing to stop the function:
typedef struct {
ebool stop_requested; // TRUE if the function must stop
} ETB_SHM_CSM_SYNCHRO_IFACE;
If the function is a long-term loop, it should be able to stop by checking the value pointed by stop_requested field of first function parameter.
This value pointed by stop_requested will be set to TRUE when the user enters a END=threadIdx

Parameters
etbpointer to etel bus structure
fctIdxthe index referencing the function (must be less than max_fct specified by function etb_csm_cmd_interface_create)
fcta pointer to the user function which will be called when user enter JMP=fctIdx,threadIdx
user_parama pointer to the user parameter which will be passed "as is" to the function
Returns
a negative value if an error or a timeout has occured
