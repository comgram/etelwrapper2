◆ dsa_create_rtv_read()

int dsa_create_rtv_read ( DSA_DEVICE *  dev,
int  reg_typ,
int  reg_idx,
int  reg_sidx,
DSA_RTV_DATA **  rtv_data 
)

Create a DSA_RTV_DATA object used for reading realtime values.

Warning: Allocate Heap memory

Get 1 or 2 free slots from UltimET and assign it or them for reading a register by sending command ASW.

Parameters
devthe drive pointer
reg_typthe register type which must be realtime-wise read
  • DMD_TYP_USER 32 bits integer user registers (X)
  • DMD_TYP_USER_INT32 32 bits integer user registers (X)
  • DMD_TYP_USER_INT64 64 bits integer user registers (XL)
  • DMD_TYP_USER_FLOAT32 32 bits float user registers (XF)
  • DMD_TYP_USER_FLOAT64 64 bits float user registers (XD)
  • DMD_TYP_PPK 32 bits integer drive parameters (K)
  • DMD_TYP_PPK_INT32 32 bits integer drive parameters (K)
  • DMD_TYP_PPK_INT64 64 bits integer drive parameters (KL)
  • DMD_TYP_PPK_FLOAT32 32 bits float drive parameters (KF)
  • DMD_TYP_PPK_FLOAT64 64 bits float drive parameters (KD)
  • DMD_TYP_MONITOR 32 bits integer monitoring registers (M)
  • DMD_TYP_MONITOR_INT32 32 bits integer monitoring registers (M)
  • DMD_TYP_MONITOR_INT64 64 bits integer monitoring registers (ML)
  • DMD_TYP_MONITOR_FLOAT32 32 bits float monitoring registers (MF)
  • DMD_TYP_MONITOR_FLOAT64 64 bits float monitoring registers (MD)
  • DMD_TYP_TRACE 32 bits integer trace buffer (T)
  • DMD_TYP_TRACE_INT32 32 bits integer trace buffer (T)
  • DMD_TYP_TRACE_INT64 64 bits integer trace buffer (TL)
  • DMD_TYP_TRACE_FLOAT32 32 bits float trace buffer (TF)
  • DMD_TYP_TRACE_FLOAT64 64 bits float trace buffer (TD)
  • DMD_TYP_LKT 32 bits integer movement lookup tables (L)
  • DMD_TYP_LKT_INT32 32 bits integer movement lookup tables (L)
  • DMD_TYP_LKT_FLOAT64 64 bits float movement lookup tables (LD)
  • DMD_TYP_TRIGGER_INT64 64 bits integer triggers buffer (EL)
  • DMD_TYP_Y 32 bits integer Y type variables (Y)
  • DMD_TYP_Y_INT32 32 bits integer Y type variables (Y)
  • DMD_TYP_ESM 32 bits integer ESM parameter (W)
  • DMD_TYP_ESM_INT32 32 bits integer ESM parameter (W)
  • DMD_TYP_ESM_INT64 64 bits integer ESM parameter (WL)
  • DMD_TYP_ESM_FLOAT32 32 bits float ESM parameter (WF)
  • DMD_TYP_ESM_FLOAT64 64 bits float ESM parameter (WD)
  • DMD_TYP_ESM_MON ESM 32 bits integer ESM monitoring (R)
  • DMD_TYP_ESM_MON ESM_INT32 32 bits integer ESM monitoring (R)
  • DMD_TYP_ESM_MON_INT64 64 bits integer ESM monitoring (RL)
  • DMD_TYP_ESM_MON_FLOAT32 32 bits float ESM monitoring (RF)
  • DMD_TYP_ESM_MON_FLOAT64 64 bits float ESM monitoring (RD)
reg_idxthe register index which must be realtime-wise read
reg_sidxthe register sub-index which must be realtime-wise read
rtv_dataa pointer to the created DSA_RTV_DATA object
Returns
a negative value if an error or a timeout has occured

DSA_EACQNOTPOSSIBLE Drives must be connected with transnet
DSA_EAXESRECOVER Transnet established but axes not recovered
