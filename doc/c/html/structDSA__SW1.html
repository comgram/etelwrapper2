Detailed Description

Etel bus drive status word 1 when acces through M60.


It can be used when status is accessed using function dsa_get_status_from_drive, because this function reads M60 and M61 of the drive.
Otherwise, this structure is obsolete and kept kept to backwards compatibility only (obsolete DSA and DSB)
EDI internal status update works now with M63. See DsaStatusDriveBitMode structure
