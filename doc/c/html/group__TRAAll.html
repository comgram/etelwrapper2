Perform download operation.

Warning: Allocate Heap memory

Parameters
traa pointer to the download traductor structure
amska mask representing all axis to download
nb_typsthe number of register types to download or -1 if all register types must be downloaded.
Must correspond to the number of element in the typs table
typsa table of all types to download. Can be empty if nb_types is -1
the following types can be downloaded
  • DMD_TYP_USER or DMD_TYP_USER_INT32 download of integer 32 bits X registers
  • DMD_TYP_USER_INT64 download of integer 64 bits XL registers
  • DMD_TYP_USER_FLOAT32 download of float 32 bits XF registers
  • DMD_TYP_USER_FLOAT64 download of float 64 bits XD registers
  • DMD_TYP_PPK or DMD_TYP_PPK_INT32 download of integer 32 bits K drive parameters
  • DMD_TYP_PPK_INT64 download of integer 64 bits KL drive parameters
  • DMD_TYP_PPK_FLOAT32 download of float 32 bits KF drive parameters
  • DMD_TYP_PPK_FLOAT64 download of float 32 bits KD drive parameters
  • DMD_TYP_MONITOR or DMD_TYP_MONITOR_INT32 download of integer 32 bits M monitoring registers
  • DMD_TYP_MONITOR_INT64 download of integer 64 bits ML monitoring registers
  • DMD_TYP_MONITOR_FLOAT32 download of float 32 bits MF monitoring registers
  • DMD_TYP_MONITOR_FLOAT64 download of float 64 bits MD monitoring registers
  • DMD_TYP_TRACE or DMD_TYP_TRACE_INT32 download of integer 32 bits T trace buffer
  • DMD_TYP_TRACE_INT64 download of integer 64 bits TL trace buffer
  • DMD_TYP_TRACE_FLOAT32 download of float 32 bits TF trace buffer
  • DMD_TYP_TRACE_FLOAT64 download of float 64 bits TD trace buffer
  • DMD_TYP_ADDRESS or DMD_TYP_ADDRESS_INT32 download of integer 32 bits A address value
  • DMD_TYP_ADDRESS_INT64 download of integer 64 bits AL address value
  • DMD_TYP_ADDRESS_FLOAT32 download of float 32 bits AF address value
  • DMD_TYP_ADDRESS_FLOAT64 download of float 64 bits AD address value
  • DMD_TYP_LKT download of movement integer 32 bits lookup tables (Only for DSCxx or DSMAXxx devices)
  • DMD_TYP_LKT_FLOAT64 download of movement float 64 bits lookup tables (Only for DSCxx or DSMAXxx devices)
  • DMD_TYP_TRIGGER download of triggers buffer
  • DMD_TYP_Y or DMD_TYP_Y_INT32 download of integer 32 bits Y type variables
  • DMD_TYP_Y_INT64 download of integer 64 bits YL type variables
  • DMD_TYP_Y_FLOAT32 download of float 32 bits YF type variables
  • DMD_TYP_Y_FLOAT64 download of float 64 bits YD type variables
  • DMD_TYP_COMMON_INT32 32 bits integer common registers (C)
  • DMD_TYP_COMMON_INT64 64 bits integer common registers (CL)

◆ tra_translate_error()

char_cp tra_translate_error ( int  code)

Convert an error code into a textual error message.

Each function of the TRA library returns an integer (int) which represents the error code. If the operation ends with success, the function returns 0, otherwise it returns the number of the error represented by a negative number. This function enables the conversion of this error code into a textual error message in the form of a character string.

Parameters
codeThe error code returned by a dsa lib function.
Returns
A pointer to a string containing the error message.
