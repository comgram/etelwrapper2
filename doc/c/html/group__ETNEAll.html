Allow to check if the custom created connection is in used.

When etne_start_custom is called, it allows another application to connect to the specified port. If this other application is connected this function will indicate it in the connected out parameter.

Parameters
busthe actually opened bus
custom_accept_portcustom port specified in etne_start_custom
connecteda boolean which will receive "true" if another application is connected to this custom port

◆ etne_translate_error()

char_cp _ETNE_EXPORT etne_translate_error ( int  code)

Convert an error code into a textual error message.

Each function of the ETNE library returns an integer (int) which represents the error code. If the operation ends with success, the function returns 0, otherwise it returns the number of the error represented by a negative number. This function enables the conversion of this error code into a textual error message in the form of a character string.

Parameters
codeThe error code returned by a dsa lib function.
Returns
A pointer to a string containing the error message.
