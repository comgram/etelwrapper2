Backup specified devices connected to specified url.

Warning: Allocate Heap memory
This function uses 7zip feature: License

Creates a .zip file which can then be downloaded into a same hardware configuration using dsa_device_configuration_download.
Search the firmwares from a sub-directory "fw" under directory of EDI-dll.
Available on Windows only.
On windows, 7z.exe and 7z.dll (provided with EDI) must be present in you PATH environment variable.
Makes first a RES.! = 0 to set the RAM with the content of the flash

Parameters
zip_file_namethe name of the created zip file
urla ETB-like urls which will be backuped
  • ETN://<hostname>[:<port>]
    Predefined ports:
    • ULTIMET-TCP/IP : 1129, 1128 or 1127
    • ACCURET through TCP/IP : 1129
  • USB
  • ULTIMET
axes_maskthe mask of the axes which must be backuped
flagthe ored value of the following flags
  • DSA_SYSTEM_CONFIGURATION_BINARY Backup binary blocks
  • DSA_SYSTEM_CONFIGURATION_TXT Backup text blocks (Will be available in the future)
avoid_block_type_maska mask of the blocks types which will not be backuped. An ored value of the following flags
  • 1 << DSA_PARAM_X_PARAMS Avoid Backup X, XL, XF, XD parameters
  • 1 << DSA_PARAM_L_PARAMS Avoid LD parameters
  • 1 << DSA_PARAM_SEQUENCES Avoid Sequences
  • 1 << DSA_PARAM_P_PARAMS Avoid P (Mapping) parameters
progress_fcta progress function which will be called (if defined) at several steps of the backup process
pusera user parameters which will be passed, as is, to the progress function
Returns
a negative value if an error occurred
